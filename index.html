What is Regions

Regions was a simple project to help create regions for a spout server, it is intended to be extended and used by other plugins by the creation of "Features". Each regions is basicly a container to hold several features that can be maintained and updated by external plugins.

Commands

The commands in regions are designed for administration and manually "fixing" any problems that might arise (plugin conflicts?). There are some very special commands that can cause alot of damage to a server if used by the wrong user, so be careful and remember, most of these commands are pretty raw and as mentioned a dozen times, ment for admin use only.

Permissions

As mentioned before, Region is designed to be extended, therefore it doesn't have all the bell and whisles to its permissions system that more detailed plugins will have. As the system was ment to be used by admins, almost all permissions are based on whether you are admin or not. While it is still possible to give permissions to regular users it is not advisable, let the plugins that extend Regions do the work.
